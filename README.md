# Fake News Detection using Machine Learning

## 📌 Description
This project detects whether a news article is real or fake using NLP and Machine Learning techniques.

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## ⚙️ Model
- Logistic Regression

## 📊 Accuracy
Achieved around 90% accuracy.

## ▶️ How to Run
pip install pandas numpy scikit-learn
python fake_news_detection.py

## 📁 Dataset
Fake.csv and True.csv
